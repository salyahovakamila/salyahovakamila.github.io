## Выбор темы оформления

Для данного проекта выбрана тема **Material for MkDocs**.

### Обоснование выбора:
1. **Современный дизайн** — чистый, минималистичный интерфейс
2. **Адаптивность** — корректное отображение на мобильных устройствах
3. **Удобная навигация** — боковое меню, вкладки, оглавление
4. **Поддержка кода** — подсветка синтаксиса для примеров


📚 Документация темы: [squidfunk.github.io/mkdocs-material](https://squidfunk.github.io/mkdocs-material/)
